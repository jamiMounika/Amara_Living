# Amara Living Showcase

A polished single-page luxury brand showcase for the Amara Living web developer internship assignment.
## 🌐 Live Demo

https://amara-living.vercel.app/


## Features

- Full-screen editorial hero with slow fading luxury interior imagery
- Featured dream room section with stone, furniture, tile, and lighting details
- Craftsmanship section for premium brand trust
- Three-category presentation for granite, furniture, and tiles
- Minimalist granite/tile space calculator with feet and meters support
- Fully responsive desktop and mobile layout
- Vanilla HTML, CSS, and JavaScript for easy deployment

## Project Structure

```text
amara-living/
├── index.html
├── css/
│   └── style.css
├── js/

└── README.md
```

## Run Locally

Open `index.html` directly in a browser, or serve the folder with any static server.

## Deploy

This project can be deployed on GitHub Pages, Netlify, or Vercel as a static site. The publish directory is the project root.
